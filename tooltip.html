Tooltip

  •    The Tooltip plugin is small pop-up box that appears when the user moves the mouse pointer over an element

Tooltip Example:

Hover over me

<a href="#" data-toggle="tooltip" title="Hooray!"><p>Hover over me</p></a>

<button type="button" class="btn btn-success" data-toggle="tooltip" title="Warning" data-placement="bottom">Hover over me!</button>



Tooltip Example:

The data-placement attribute specifies the tooltip position.

  • Top

  • Bottom

  • Left

  • Right




<div class="container">
<p>The data-placement attribute specifies the tooltip position.</p>
<ul class="list-inline">
  <li><a href="#" data-toggle="tooltip" data-placement="top" title="Top!">Top</a></li>
  <li><a href="#" data-toggle="tooltip" data-placement="bottom" title="Bottom!">Bottom</a></li>
  <li><a href="#" data-toggle="tooltip" data-placement="left" title="Left!">Left</a></li>
  <li><a href="#" data-toggle="tooltip" data-placement="right" title="Right!">Right </a></li>
</ul>
</div>
<script>
   $(document).ready(function(){
   $('[data-toggle="tooltip"]').tooltip();
});
</script>
