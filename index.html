What is Bootstrap?

  • front-end framework
  • Free and open source
  • coded in CSS, Javascript.
  • Widely used for creating mobile first website.





Why to use bootstrap??

  • Open-Source
  • Easy to use
  • Responsive features
  • Mobile-first approach





How to use Bootstrap

Download Bootstrap OR give link

  • Download Bootstrap:
    • Download from http://getbootstrap.com
    • Give link of downloaded bootstrap files
      <link href="css/bootstrap.min.css" rel="stylesheet">
      <script src="js/jquery-2.2.1.min.js"></script>
      <script src="js/bootstrap.min.js"></script>

  • Online links of bootstrap
    <link rel="stylesheet" href="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css">

    <!-- jQuery library -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>

    <!-- Latest compiled JavaScript -->
    <script src="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/js/bootstrap.min.js"></script>

  • Now You are ready to use each and every feature of Bootstrap :-)
