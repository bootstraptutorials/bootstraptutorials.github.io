Buttons in Bootstrap



Add below classes to anchor tag or button tag

  • btn btn-default
  • btn btn-primary
  • btn btn-success
  • btn btn-info
  • btn btn-warning
  • btn btn-danger

These classes are known as Contextual Class


button with links:


Default Primary Success Info Warning Danger
<a href="index.html" class="btn btn-default">Default</a>
<a href="#" class="btn btn-primary">Primary</a>
<a href="#" class="btn btn-success">Success</a>
<a href="#" class="btn btn-info">Info</a>
<a href="#" class="btn btn-warning">Warning</a>
<a href="#" class="btn btn-danger">Danger</a>






button Group:

  • .btn-group
  • .btn-group-vertical
  • .btn-group-justified
